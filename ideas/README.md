# Ideas

Esta carpeta contiene todas las ideas de nuevos productos o features para productos existentes.

## Plantillas Disponibles

### TEMPLATE.md - Productos y Features Generales
Para nuevos productos o features de productos existentes.

### TEMPLATE-AGENTE-IA.md - Agentes de IA
Para automatizaciones con agentes autónomos de IA. Incluye:
- Definición de inputs/outputs
- Comportamiento y prompts
- Herramientas y capacidades
- Flujo de trabajo detallado
- Casos de uso específicos
- Criterios de éxito

## Cómo Proponer una Idea

### Para Productos/Features:
1. Copia el archivo `TEMPLATE.md`
2. Renómbralo con un nombre descriptivo: `mi-idea-descriptiva.md`
3. Completa todas las secciones relevantes
4. Guarda el archivo en esta carpeta

### Para Agentes de IA:
1. Copia el archivo `TEMPLATE-AGENTE-IA.md`
2. Renómbralo: `agente-[nombre-descriptivo].md`
3. Completa todas las secciones (especialmente prompts y herramientas)
4. Guarda el archivo en esta carpeta

## Estructura de una Idea

- **Resumen Ejecutivo**: ¿Qué es?
- **Problema**: ¿Qué resuelve?
- **Solución**: ¿Cómo lo resuelve?
- **MVP**: ¿Qué funcionalidades mínimas necesita?
- **Casos de Uso**: ¿Cómo se usará?
- **Criterios de Éxito**: ¿Cómo medimos el éxito?

## Ideas para Productos Existentes

Si tu idea es un feature para un proyecto existente, usa este formato en el nombre:

```
[nombre-proyecto]-feature-[descripcion].md
```

Ejemplo: `crm-feature-reportes-avanzados.md`

## Siguiente Paso

Una vez documentada la idea:
1. El agente de producto la revisará
2. Generará un plan de desarrollo
3. Propondrá una arquitectura
4. Creará un prototipo funcional en `/prototipos`
